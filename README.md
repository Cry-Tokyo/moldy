# Moldy
Moldy is a Libary for Managing Downloads, written in Rust.

Features :
- [ ] Provide a CLI and GUI
- [ ] Checksum verification
- [ ] Web browser and File browser integration
- [ ] Resuming
- [ ] Download acceleration
- [ ] URLs per file
- [ ] Mirror search
- [ ] Auto dial/hangup and Scheduler
- [ ] Cookies import
- [ ] Speed limit
- [ ] Categorized downloads
- [ ] Metadata preservation
- [ ] Add-on support
- [ ] Remote usage
Support :
- [ ] HTTPS and HTTP
- [ ] SFTP and FTP
- [ ] Magnet, BitTorrent, and RSTP
- [ ] Tor and I2P
- [ ] Legacy Protocols: RTMP, MMS, eDonkey, Metalink, and RSS
